# elokuvasovellus

sovelluksen toiminnot:

-Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.

-käyttäjä voi lisätä sovellukseen elokuvan nimen ja arvostelemaan sen asteikolla 1-10, sekä lisäämään kirjallisen arvion.

-käyttäjä voi lisätä ilmoitukseen kuvan.

-käyttäjä näkee muiden ilmoitukset sekä lisäämään arvioita niihin

-käyttäjä pystyy etsimään alokuvia nimen sekä genren perusteella.

-käyttäjä voi luokitella elokuvan eri genrejen mukaan

-Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja ja käyttäjän lisäämät ilmoitukset.
